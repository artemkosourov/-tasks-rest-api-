# Tasks REST API

REST API для управления задачами с аутентификацией пользователей и интеграцией с Kafka.

## 🚀 Технологии

- **Go** 1.25
- **PostgreSQL** 16
- **Kafka** (с Zookeeper)
- **Docker** & Docker Compose
- **JWT** для аутентификации
- **Swagger** для документации API

## 📋 Требования

- Docker и Docker Compose
- Go 1.25 (для локальной разработки)
- Make (опционально)
